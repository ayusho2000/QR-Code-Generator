# QR-Code-Generator

The **QR Code Generator** is a Node.js-based command-line application that allows users to generate QR codes from any text or URL input.

It provides a fast and simple way to convert user input into a QR image while also storing the input for future reference.

---

##  Features

###  User Input
- Accepts input via terminal using interactive prompts  

###  QR Code Generation
- Converts text/URL into QR code image  
- Generates image instantly  

###  Data Storage
- Saves user input into a `.txt` file  

---

##  Tech Stack

### Backend
- Node.js  

### Libraries
- inquirer  
- qr-image  

### Core Module
- fs (File System)  

---

##  System Workflow

1. User enters text or URL  
2. Application processes input  
3. QR code image is generated  
4. Input is saved in a text file
